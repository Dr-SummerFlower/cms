# 用户模块 (users)

上级模块说明见 `../AGENTS.md`（src）。

## 职责

用户信息查询与更新、实名信息、角色管理，以及管理端用户列表与操作。

## 主要文件

- **users.controller.ts** — 用户信息、更新、管理端列表等接口。
- **users.service.ts** — 用户与实名等业务逻辑。
- **dto/** — 分页、更新用户、角色、列表响应等 DTO。
- **entities/** — 用户实体、实名实体（Mongoose schema）。

## 约定

- 与 auth 模块配合：认证后的用户标识用于本模块查询与更新。
- 管理端列表与角色变更与前端 `cms_client` 管理后台用户管理对应；敏感字段不返回或脱敏。
