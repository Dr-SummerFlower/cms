# 工具函数 (utils)

上级目录说明见 `../AGENTS.md`（src）。

## 职责

与业务无强绑定的通用工具：HTTP 客户端、认证辅助、Cookie、图片处理等。

## 当前文件

- **http.ts** — Axios 实例、请求/响应拦截、统一错误处理与 baseURL。
- **auth.ts** — 与登录态相关的工具（如 Token 读写、登出）。
- **cookie.ts** — Cookie 的读写封装。
- **image.ts** — 图片相关工具（如压缩、预览）。

## 约定

- API 请求统一经 `http.ts` 发出，以便统一处理认证头、错误与刷新逻辑。
- 不在此处写业务逻辑，仅提供可复用的底层能力。
